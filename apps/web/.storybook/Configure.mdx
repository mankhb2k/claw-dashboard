      ### 🛠 Technology Stack
      Built on a modern stack that prioritizes flexibility and performance:
      - **Framework**: Next.js 16 (App Router)
      - **Styling**: Vanilla CSS Modules (no utility framework)
      - **Icons**: Lucide React / React Icons
      - **State**: Zustand
      - **Validation**: Zod
    </div>

    <div className="sb-section-item">
      ### 📜 Core Principles (Rule.md)
      Every UI component in the project must follow these rules:
      - **No Tailwind/Bootstrap**: Write custom CSS to maintain full control over the interface.
      - **CSS Variables**: Always use system variables for colors and spacing.
      - **Atomic Design**: Prefer using and extending components in `components/ui/`.
      - **Consistency**: Hover and focus effects must be consistent across the entire system.
    </div>
  </div>
</div>
