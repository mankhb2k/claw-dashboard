# Chat UI

Stories for the realtime chat flow ‚Äî tool events, markdown bubbles, and live thread.

## Story groups

| Folder | Component | Description |
|---|---|---|
| **ContentArea** | `Chat/ContentArea` | Scrollable thread preview (no composer) |
| **ToolActivity** | `ToolActivityCard` | Per-tool call card (exec, read, MCP, ‚Ä¶) |
| **ToolActivity** | `ToolActivityBar` | Inline status bar / preparing state |
| **ToolActivity** | `ToolResearchBlock` | Web search / fetch group + sources |
| **ToolActivity** | `ChatLiveThread` | Orchestrator ‚Äî text + tools in one thread |
| **Message** | `ChatMessageBubble` | User / assistant bubble (markdown) |

## Controls

Shared args live in `components/chat/tool-activity.story-controls.ts`:

| Control | Used in | Effect |
|---|---|---|
| `toolPreset` | Card, Bar, LiveThread, ContentArea | Tool type ‚Üí i18n label |
| `status` | Card, Bar, LiveThread | running / done / error label |
| `withArgs` / `withOutput` | Card | Expanded body content |
| `showPreparing` | Bar, LiveThread | Preparing row |
| `researchPreset` | ResearchBlock, LiveThread | Web search flow |
| `scenario` | ContentArea | Full thread preset |

Use **Playground** to tweak controls; **AllToolLabels** / **AllStatuses** show every label variant.

## Run Storybook

```bash
cd aucobot
pnpm --filter @aucobot/web run storybook
```

Open the **Chat** sidebar ‚Üí pick a component. Use the **Theme** toolbar to switch light/dark.
