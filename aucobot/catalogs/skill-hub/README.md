# Local skill-hub catalogs (offline dev)

Sample `SKILL.md` packages under this folder are **not wired** to the production API.
Runtime skill installation uses [ClawHub](https://clawhub.ai) via `@aucobot/clawhub-client`.

Use these folders only for:

- Manual inspection of OpenClaw skill package layout
- Future offline dev seed scripts (not implemented)

To install skills in the app, use **Dashboard → Skills → Browser Store** or create a project skill via the API.
