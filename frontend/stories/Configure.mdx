      - **Framework**: Next.js 16 (App Router)
      - **Styling**: Vanilla CSS Modules (Không dùng Framework)
      - **Icons**: Lucide React / React Icons
      - **State**: Zustand
      - **Validation**: Zod
    </div>

    <div className="sb-section-item">
      ### 📜 Nguyên tắc cốt lõi (Rule.md)
      Mọi thành phần UI trong dự án phải tuân thủ các quy tắc nghiêm ngặt:
      - **No Tailwind/Bootstrap**: Tự viết CSS để kiểm soát 100% giao diện.
      - **CSS Variables**: Luôn sử dụng biến hệ thống cho màu sắc và khoảng cách.
      - **Atomic Design**: Ưu tiên sử dụng và mở rộng các component trong `components/ui/`.
      - **Consistency**: Hover và Focus effect phải đồng nhất trên toàn hệ thống.
    </div>
  </div>
</div>

## 📐 Khoảng cách & Hình khối (Spacing & Shapes)

| Biến số | Giá trị | Mô tả |
|---|---|---|
| `--radius-md` | 10px | Bo góc trung bình |
| `--radius-lg` | 12px | Chuẩn bo góc của Card |
| `--space-4` | 1rem (16px) | Khoảng cách padding/margin tiêu chuẩn |
| `--transition-base` | 180ms | Tốc độ hiệu ứng mặc định |
