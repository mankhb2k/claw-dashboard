## Bug đã fix trong run này

### BUG-001 — Google redirect trả 200 thay vì 302
- **Phát hiện:** T9 trả HTTP 200 với Location header thay vì HTTP 302
- **Nguyên nhân:** Fastify v5 thay đổi `reply.redirect()` — không tự set status code nữa
- **Fix:** `reply.status(302).header('Location', url).send()` tại `signInGoogle()` và `googleCallback()`
- **File:** `src/auth/auth.controller.ts`
